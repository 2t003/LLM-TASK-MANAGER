# LLM Task Manager

Gestionnaire de tâches piloté par LLM (squelette de projet).

## Structure

Voir `ARCHITECTURE.md` pour une description détaillée de l’architecture des dossiers.

## Prérequis

- Python 3.10+ (recommandé)

## Installation (exemple)

```bash
cd llm-task-manager
python -m venv .venv
source .venv/bin/activate  # sous Windows: .venv\\Scripts\\activate
pip install -e .
```

## Lancer l’application (placeholder)

```bash
python -m app.main
```

> Pour l’instant, ceci ne fait qu’afficher un message de squelette dans la console.
